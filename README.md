# easylang

## Description
easyLang is a conlang (Constructed Language) organizer website made for particularly indecisive conlangers. 

At the website's core is the **Dictionary** and **Sentences**. 

The *Dictionary* allows users to input words for their langauge and group them into the part of speech that they are.

The *Sentence* page allows users to arrange their words in order to create a cohesive sentence. When creating sentences the user selects from words defined in their dictionary. Then once the user defines the meaning of the sentence, the user can hover over each word to find out what the word means without looking in the dictionary.

Users can also create an account to work on other devices.

## Project Makeup
easyLang is made with [Vue.js](https://vuejs.org)  and [Firebase](https://firebase.google.com)
